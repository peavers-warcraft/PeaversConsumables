# PeaversConsumables

Shows the best consumables, enchants and gems for your class and spec — click any item to search for it on the Auction House.

[peavers.io](https://peavers.io) | [Report Issues](https://github.com/peavers-warcraft/PeaversConsumables/issues)

## Features

- Best enchants, gems and consumables for your current spec, sourced from Wowhead
- Click any item to instantly search for it on the Auction House
- Window opens automatically alongside the Auction House (configurable)
- Data updates automatically via the PeaversConsumablesData companion addon

## Installation

Install from CurseForge. Requires:

- [PeaversCommons](https://www.curseforge.com/wow/addons/peaverscommons)
- [PeaversConfig](https://www.curseforge.com/wow/addons/peaversconfig)
- [PeaversConsumablesData](https://github.com/peavers-warcraft/PeaversConsumablesData)

## Usage

- `/pcons` — toggle the consumables window
- `/pcons config` — open configuration (also available via `/pconfig`)
- `/consumables` — alias for `/pcons`

Open the Auction House and the window appears next to it; click an item to search for it in the browse tab.
